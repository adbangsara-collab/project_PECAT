README

P.E.C.A.T or also known as Personal Enlightening Computer Academic Trainer is a computer program made to serve the purpose of being a learning resource including topics from the Philippine Science Highschool 2025 Grade 8 curriculum. Please note that this program is currently in its early stages of development, meaning that its contents are limited. It is planned that for the Alpha stage of this program, only lessons from Biology will be included for the meanwhile until further notice. P.E.CA.T includes different kinds of learning resources that the user can choose from based on their personal preference.
